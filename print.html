Shortcuts

Shortcuts 概述

快捷键，又称为快速键、组合键或热键，指通过某些特定的按键、按键顺序或按键组合来完成一个操作，很多快捷键往往与如 Ctrl 键、Shift 键、Alt 键、Fn 键以及 Windows 平台下的 Win 键和 Mac 机上的 ⌘ Cmd 鍵（曾有Meta键）等配合使用。

这个项目是以提高自己工作效率为目的，将工作中常用到的工具或者软件的快捷键总结下来，列个清单方便以后查找。

Oh My Zsh 是一个令人愉悦的、开源的、社区驱动的框架，用于管理您的 Zsh 配置。它捆绑了成千上万个有用的功能、助手、插件、主题以及一些让您兴奋的东西...

git plugin

The git plugin provides many aliases and a few useful functions.

To use it, add git to the plugins array in your zshrc file:

plugins=(... git)

Git-Flow plugin

This plugin adds completion and aliases for the git-flow command.

To use it, add git-flow to the plugins array in your zshrc file:

plugins=(... git-flow)
